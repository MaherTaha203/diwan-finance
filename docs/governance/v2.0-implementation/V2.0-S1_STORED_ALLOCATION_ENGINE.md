## Constitutional interpretation applied (documented, not invented)
Under the continuous running balance (CA‑006), **"Historical Debt" = all debt from before the current
operating year** (the aggregate opening balance plus any prior‑year unpaid dues), making Current /
Historical / Future exhaustive. This is the plain reading of "Historical Debt" in a running‑balance model
where "previous debt remains part of the same running balance" (OD‑06). If the owner intends prior‑year
unpaid dues to be a **separate** category from the opening balance, that is a one‑line constitutional
clarification; the flag‑OFF engine changes nothing until then.
