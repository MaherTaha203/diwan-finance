# Architecture Decision Records (ADR) — Register

The platform's ADR register. An **ADR** is mandatory before work proceeds whenever GOV‑WS‑01
v1.5 **§6** requires one; its format is P‑AIENG‑S2 **RS‑ADR** (id · context · decision ·
alternatives · consequences · status). Each ADR is `ADR‑<scope>‑NN`, lives in this directory,
and is **owner‑ratified at merge**. The process was adopted by ADR‑GOV‑01.
