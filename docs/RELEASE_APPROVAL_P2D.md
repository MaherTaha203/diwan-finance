## القسم 1 — النطاق (قدرات العمل التي تغيّرت في P2)

1. **ثلاث خزائن نقدية حقيقية** — الغداء، الديوان، العجز التاريخي — بدل صندوقين + «وعاء تبرعات» وهمي.
2. **سجلّان للتبرعات** — سجل نقدي مرجعي (التبرع النقدي حركة في خزينته + قيد مرجعي) وسجل عيني/خدمي
   توثيقي **لا يملك رصيداً ولا يمسّ نقداً**.
3. **تصنيف صريح لكل حركة مالية** (نوع + وجهة + سبب) — والسندات التاريخية أُعيد تصنيفها بسجل
   قرارات موقَّع من المالك **دون مساس بأي رقم أو تاريخ أو مبلغ أو نيّة**.
4. **الإدخال لا يخمّن**: التبرّع الجديد يتطلب نوعاً صريحاً (نقدي/عيني-خدمي) ووجهة صريحة.
5. **قاعدة الفائض**: بلوغ العجز صفراً يحوِّل الفائض تلقائياً لخزينة الغداء.
6. **قفل التصنيف**: التصنيف تاريخ مالي؛ الاستثناء الوحيد إداري بسبب إلزامي وأثر تدقيق كامل.
7. **أدلّة دائمة**: الملحق أ، تقارير الترحيل والتحقّق، خطوط أساس مالية لكل مرحلة.

## القسم 4 — تحقّق المستخدم

**PASS.** المدير العادي يرى أربعة تبويبات بأسماء صريحة، وكل بطاقة تعرض **قاعدتها الحسابية
بلغة عادية** (العجز: «الأصلي + التحصيل − التسويات؛ الفائض للغداء»؛ السجلّات: «لا تمثّل نقداً»)،
والإدخال يسأله مباشرة: نقدي أم عيني؟ وإلى أين؟ — لا مفاهيم مزدوجة ولا «وعاء» غامض.
